# Modern Flutter UI Trends (2025-2026)

- **Glassmorphism:** 배경 블러(`BackdropFilter`)와 반투명 레이어를 적극 활용.
- **Bento Grids:** 정보를 직사각형 그리드 시스템으로 구조화.
- **Micro-Interactions:** 버튼 클릭 시 `Scale`, `HapticFeedback` 필수.
- **Typography:** 큰 헤드라인과 넉넉한 Whitespace. `GoogleFonts` 활용.
- **Motion:** 화면 전환 시 `OpenContainer` 또는 `SharedAxis` 전환 효과 사용.
